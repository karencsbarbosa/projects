IF Statement

The if statement in C# is used to execute a block of code if a specified condition is true. It allows developers to create decision-making logic within their programs.

The syntax typically consists of the if keyword followed by the condition to be evaluated and the code block to execute if the condition is true. Example:

int x = 10; if (x > 5) { Console.WriteLine("x is greater than 5"); }
  • The if statement checks whether a condition is true.
  • If the condition is met, the code block within the if statement is executed.
FOR Loop

The for loop in C# is a control flow statement used to execute a block of code a specified number of times. It consists of three parts: initialization, condition, and iteration.

This loop is particularly useful when the number of iterations is known beforehand. Example:

for (int i = 0; i < 5; i++) { Console.WriteLine("Iteration: " + i); }
  • The for loop executes a block of code a specified number of times.
  • It consists of initialization, condition, and iteration parts.
SWITCH Statement

The switch statement in C# provides a way to perform different actions based on the value of a variable or expression. It allows developers to write more concise and readable code when dealing with multiple conditional cases.

The switch statement evaluates the value of an expression and executes the corresponding case. Example:

int day = 3; switch (day) { case 1: Console.WriteLine("Monday"); break; case 2: Console.WriteLine("Tuesday"); break; default: Console.WriteLine("Other day"); break; }
  • The switch statement performs actions based on the value of a variable.
  • It provides a concise way to handle multiple conditional cases.
WHILE Loop

The while loop in C# is used to execute a block of code repeatedly as long as a specified condition is true. It is particularly useful when the number of iterations is not known beforehand.

The loop evaluates the condition before executing the code block. Example:

int count = 0; while (count < 5) { Console.WriteLine("Count: " + count); count++; }
  • The while loop executes a block of code while a condition is true.
  • It is useful when the number of iterations is not known beforehand.
Method Declaration

Method declaration in C# defines a reusable block of code that performs a specific task. It encapsulates functionality, promoting code reusability and maintainability.

Methods can accept parameters and return values, allowing for flexible usage across different parts of a program. Example:

int Add(int a, int b) { return a + b; }
  • Methods encapsulate a set of instructions that perform a specific task.
  • They enhance code reusability and maintainability.
